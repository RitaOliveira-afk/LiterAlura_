<h2>Challenge</h2>

  <p>
    Desafio LiterAlura é um projeto onde é possível realizar consultas de livros e dados de autores, por meio de consumo de <strong>API</strong> e conexão a um <strong>Banco de Dados</strong> relacional.
    Esse projeto faz parte da formação em java do programa ONE com a Alura + Oracle.
  </p>
  
  <p>
    <strong>⚠️ Observação:</strong> Uso da Api Gutendex que é baseado no Project Gutenberg, que é majoritariamente em inglês.
  </p>


<h2>Objetivos do projeto</h2>
<ul>
  <li>Buscar Livro pelo Título</li>
  <li>Listar livros registrados</li>
  <li>Listar livros registrados/li>
  <li>Listar autores vivos em um determinado ano</li>
   <li>Listar livros em um determinado idioma</li>
   <li>Aplicar conhecimentos em Java e Spring, como consumo de APIs externas, persistência de dados e etc.</li>
  
</ul>

<h2>Tecnologias utilizadas</h2>
<ul>
  <li>Java</li>
    Spring Boot
    Spring Data JPA
    Hibernate
    PostgreSQL
    Maven
    API Gutendex
    Git e GitHub</li>
  

</ul>
